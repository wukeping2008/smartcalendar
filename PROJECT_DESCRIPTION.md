# Smart Calendar Assistant

**A revolutionary AI-powered calendar and schedule planning assistant with fluid time management interface.**

## 🌟 Key Highlights

- **🌊 TimeFlow Management**: Revolutionary fluid interface where events flow like water with drag-and-drop scheduling
- **🤖 AI Voice Assistant**: Natural language processing with Coze API for intelligent conversation
- **📅 Smart Calendar**: Traditional calendar view with advanced event management
- **🎨 Modern UI**: Beautiful gradient design with smooth animations and responsive layout

## 🚀 Technology Stack

- **Frontend**: Vue 3 + TypeScript + Pinia
- **AI Integration**: Coze API + WebSocket + Web Audio API
- **Animation**: Canvas API for fluid TimeFlow rendering
- **Storage**: JSON-based lightweight data persistence

## 📱 Features

### TimeFlow Interface
- Drag events to reschedule with visual feedback
- 10%-300% zoom for precise time management
- Real-time conflict detection
- AI-powered scheduling suggestions

### Voice Assistant
- Natural voice commands for scheduling
- Real-time audio streaming
- Multimodal text + voice interaction
- Intelligent calendar management

### Traditional Calendar
- Month/week/day views
- Event categories and priorities
- Quick event creation
- Advanced filtering options
