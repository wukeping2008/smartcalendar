# 🌊 时间流管理 - 用户指南

## 功能概述

时间流管理是一个革命性的时间可视化工具，让您的日程安排如水流般自然流动。

## 🎯 核心功能

### 1. 拖拽操作
**如何拖拽事件：**
- 点击并按住任意事件卡片
- 拖拽到新的时间位置
- 松开鼠标完成移动

**拖拽提示：**
- 🟢 绿色区域：可以安全放置
- 🔴 红色区域：存在时间冲突
- 📍 网格吸附：自动对齐到30分钟间隔
- 👻 半透明预览：显示事件将要放置的位置

### 2. 缩放功能
**缩放控制：**
- 使用左上角的缩放滑块
- 范围：10% - 300%
- 实时显示当前缩放比例

**缩放效果：**
- 放大：查看更多细节，精确调整时间
- 缩小：查看全天概览，快速规划

### 3. 时间导航
**快速定位：**
- 📍 "现在" 按钮：立即跳转到当前时间
- 🕐 时间轴：点击任意时间快速跳转
- ⏰ 自动居中：新事件自动显示在可视区域

### 4. 动画控制
**流体动画：**
- ▶️ 播放：启用流体动画效果
- ⏸️ 暂停：停止动画，静态显示
- 🌊 波纹效果：事件移动时的水波动画

## 🎨 视觉指南

### 事件颜色编码
- 🔵 **蓝色**：工作任务
- 🟣 **紫色**：会议
- 🟢 **绿色**：个人事务
- 🟠 **橙色**：休息时间

### 优先级标识
- 🔴 **红色边框**：高优先级
- 🟡 **黄色边框**：中优先级
- 🟢 **绿色边框**：低优先级

### 状态指示
- ✨ **发光效果**：当前选中的事件
- ⚠️ **红色闪烁**：存在冲突的事件
- 👆 **鼠标悬停**：显示详细信息

- `+/-`：放大/缩小

### 鼠标操作
- **单击**：选择事件
- **双击**：编辑事件
- **右键**：显示上下文菜单
- **拖拽**：移动事件
- **滚轮**：缩放视图

## 💡 智能功能

### 冲突检测
系统会自动检测以下冲突：
- ⏰ **时间重叠**：两个事件时间冲突
- ⚡ **能量冲突**：高强度任务安排过密
- 🎯 **优先级冲突**：重要任务时间不合理

### 智能建议
- 📊 **最佳时间**：根据您的习惯推荐最佳时间
- ☕ **休息提醒**：自动建议添加休息时间
- 🔄 **重新排列**：一键优化全天安排

## 🛠️ 常见问题

### Q: 为什么拖拽没有反应？
A: 请确保：
1. 事件已被选中（有发光边框）
2. 拖拽距离超过5像素阈值
3. 动画功能已启用

### Q: 如何精确调整时间？
A: 
1. 放大视图到200%以上
2. 启用网格吸附功能
3. 使用详情面板的时间输入框

### Q: 冲突如何解决？
A: 
1. 查看冲突面板的建议
2. 点击"解决"按钮自动调整
3. 手动拖拽事件到合适位置

### Q: 动画太快/太慢怎么办？
A: 在设置面板中调整动画速度和流体效果强度

## 🎯 最佳实践

### 高效使用技巧
1. **先规划后细化**：先用缩小视图规划全天，再放大调整细节
2. **颜色分类**：为不同类型的事件设置不同颜色
3. **优先级排序**：重要事件安排在精力最好的时间段
4. **留白时间**：在事件间留出缓冲时间

### 避免常见错误
- ❌ 不要将事件拖拽到时间轴上
- ❌ 不要忽视冲突警告
- ❌ 不要在缩放过小时进行精确操作
- ❌ 不要关闭网格吸附功能
