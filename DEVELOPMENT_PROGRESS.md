## 📝 开发笔记

### 最佳实践
1. 使用 TypeScript 严格模式
2. 组件保持单一职责
3. 使用 React.memo 优化渲染
4. 状态管理使用 Zustand
5. 样式使用 Tailwind CSS

### 注意事项
- AI API 调用需要限流
- IndexedDB 需要定期清理
- 3D 渲染注意内存管理
- WebSocket 需要重连机制
